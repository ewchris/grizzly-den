About Me

I am a software developer that has been working in cloud infrastructure management for 7 years. I create front-end tools to allow users to manage servers hosted on AWS and Azure. Tracking and managing server, product, user, and environment states, and building experiences that ensure everything is handled with care, is a big part of my everyday work.

Another big part of my day is helping to improve my team as its scrum master. Making sure to create an open environment for reliable, consistent, and productive communication is a major factor in building a high performing team.

The technology that helped to build this site is the same tech I use daily. I use Angular with Tailwind, and Jest for testing; all written using Typescript. This site is currently hosted in AWS, while in my day to day work, I am using the full suite of features from AWS and Azure, that complexity isn't needed for this project ;P, so here I am using the Amplify platform to allow me to get this project up and running quickly while keeping costs low.

This project is a little sandbox for me play with the tools and tech I love, while also trying out some new stuff to keep my skills current and being a creative outlet without the constraints and complexity of enterprise level development.
